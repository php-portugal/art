### Meetups

Editable meetup banner in .svg format.

#### How to use

1. Open the meetup banner in your favorite code editor
2. Edit the text tags with the meetup location and date.
3. Replace the image with the one you want to use.
4. Output
   1. You may use the SVG as is.
   2. You may export it as a PNG or JPG.

#### Exporting to PNG with Google Chrome

1. Open the meetup banner in Google Chrome
2. Inspect the page
3. Right click the svg node
4. Select "Capture node screenshot"
5. Use!
